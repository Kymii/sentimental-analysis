# sentimental-analysis

**Description**

A web tool that allows users to run natural language processing (NLP) on articles on blogs found on other websites.

**npm run build-prod:** This command builds app for production to the dist folder

**npm run build-server:** This command starts the server

**npm run build-client:** This command runs the app in development and automatically opens localhost:8080 in the browser

**npm run jest:** This command runs all jest test cases
